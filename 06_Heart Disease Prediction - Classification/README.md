# ❤️ Heart Disease Prediction App

A Machine Learning web application built with **Streamlit** that predicts whether a person is at **high** or **low risk** of heart disease based on medical information.

## 🚀 Features

- Predicts heart disease risk in real time
- Simple and interactive Streamlit interface
- Uses the trained **multiple** model

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

## 📂 Project Structure

```
├── app.py              # Streamlit application
├── Heart.ipynb         # Model training and analysis
├── KNN_heart.pkl       # Trained KNN model
├── scaler.pkl          # StandardScaler
├── columns.pkl         # Feature columns
├── heart.csv           # Dataset
└── README.md
```

## 📊 Input Features

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

## 📈 Output

The application predicts:

- ✅ Low Risk of Heart Disease
- ⚠️ High Risk of Heart Disease
